# Deep Thinking

**Topic:** $ARGUMENTS

## Modes

### Problem Solving (default)
```bash
/think "Should we use microservices?"
/think "How to scale to 10x traffic?"
```

### Brainstorming
```bash
/think brainstorm "Dashboard redesign ideas"
```

## Analysis Framework

### 1. Technical Perspective
- Feasibility, constraints
- Performance, scalability
- Security implications
- Technical debt

### 2. Business Perspective
- Value, ROI
- Time-to-market
- Competitive advantage
- Risk vs reward

### 3. User Perspective
- User needs, pain points
- Usability, accessibility
- Edge cases

### 4. System Perspective
- Integration points
- Dependencies
- Emergent behaviors

## Thinking Principles
- **First Principles**: Break to fundamentals
- **Inversion**: What to avoid?
- **Second-Order**: Consequences of consequences
